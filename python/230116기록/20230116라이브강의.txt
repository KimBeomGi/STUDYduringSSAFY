프로그래밍이란?
소프트웨워=컴퓨터에게 일시키는 도구
소프트웨어를 개발하기 위한과정(코딩)
컴퓨터에게 명령하는 '적절한 수행절차를 정의'하고 이를 '프로그래밍 언어로 표현'

프로그래밍 과정
1. 컴퓨터에게 시키고 싶은 일을 정한다.
2. 컴퓨터가 이해할 수 있도록 수행 절차를 정의해서 표현
3. 적절한 프로그래밍 언어를 선택, 언어를 이용해서 절차를 기술
4. 발생하는 오류를 수정
  4-1. 구문 오류 → 문법오류 실행안됨
  4-2. 논리 오류 → 문법됨 실행 됨 결과가 이상함

컴퓨팅 사고력
1. 컴퓨터의 특성을 잘 이해한다.
2. 문제해결능력을 기른다.
3. 프로그래밍 언어에 능숙해진다.(많이 해보면 는다.)

*PYTHON*
기계어의 대안으로 사람이 이해할 수 있는 새로운 언어->프로그래밍 언어

소스코드: 프로그래밍 언어로 작성된 프로그램
번역기(interpreter or compiler)

  - 파이썬의 경우 interpreter로 사용가능

변수(variable)

변수 할당연산자 값
변수란
-데이터를 저장하기 위해 사용
-변수를 사용하면 복잡한 값들을 쉽게 사용할 수 있음

-이후 코드를 수정하기 용이함
-코드의 가독성 증가

변수의 할당
-할당 연산자를 통해 값을 할당
-같은 값을 동시에 할당 가능
-다른 값을 동시에 할당 가능

예)임시변수를 이용한 값 교체

식별자(Identifiers)	(변수명은 변수의 의미를 알 수 있게 작성해보자.)
-변수의 이름을 식별자라고 함(변수,함수,클래스...)
-'변수명 짓기 사이트'도 있다
- 읽기 쉽고 이해하기 쉽게 변수명작성

변수 이름 규칙
-식별자의 이름은 영문 알파벳, 언더스코어(_), 숫자로 구성
-첫 글자에 숫자 불가
-길이제한 없고 ,대소문자를 구별
-다음의 키워드는 예약어로 사용X(False, True ,,,,)
-내장 함수나 모듈 등의 이름도 사용X

-유지보수 용이

*자료형

프로그래밍에서 변수는 메모리의 주소를 기억하는 이름!
-우리는 변수를 이용해서 데이터를 기억
-자료형마다 차지하는 메모리의 크기가 다르다.
파이썬에서 id(보고싶은 변수) 메모리 어떤주소에 저장되어있는지 주소를 볼 수 있음
-이것을 알려주는 이유는 나중에 코드가 길어질 때 해당 변수가 동일한 데이터인지 확인하기 위함임

자료형 분류
-수치형
  -int(정수자료형)
  -float(실수자료형)
    -실수 연산시 주의할점(부동 소수점)
-문자열
  -str(모든 문자는 str타입)
    -중첩 따옴표(따옴표 안에 따옴표를 표현) " '' " 또는 ' "" '로 표현하면 된다.
    -삼중 따옴표
      -작은따옴표나 큰 따옴표를 삼중으로 사용(따옴표 안예따옴표를 넣을 때, 여러줄을 나눠 입력할때 편리)
  -Escape sequence
    -역슬래시 뒤에 특정 문자가 와서 특수한 기능을 하는 문자 조합
        -\r 캐리지리턴은 터미널에서 깜빡거리는 커서뒤의 것을 제일앞에서부터 채워넣음('aaaa\rbbb' → bbba 이 된다.)
  -string Interpolation(문자열을 변수를 활용하여 만드는 법
-불린형
-None
